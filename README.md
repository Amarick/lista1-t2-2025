# Lista 1 - Codifica (2025)

Este repositório contém a primeira lista de exercícios desenvolvida durante o curso da **Codifica** – Turma 2 de 2025.  
Os programas foram escritos em **JavaScript** e executados via **Node.js** no terminal.

## 📁 Estrutura

Os arquivos estão organizados na pasta `exercicios/`, contendo os 15 exercícios numerados de `exe1.js` a `exe15.js`.

## ⚙️ Pré-requisitos

Para executar os exercícios localmente, você precisa ter instalado:

- [Node.js](https://nodejs.org/) (versão recomendada: 18 ou superior)
- Git (opcional, para clonar o repositório)

## 💻 Como configurar o projeto

1. **Clone este repositório:**

```bash
git clone https://github.com/Amarick/lista1-t2-2025.git
cd lista1-t2-2025
```
2. **Instale as dependências:**

```bash
npm install
```

> A dependência utilizada é:
> `prompt-sync` — para entrada de dados via terminal.

3. **Verifique se o Node.js está instalado:**

```bash
node -v
```

Se não estiver instalado, baixe e instale a partir do site oficial: [https://nodejs.org/](https://nodejs.org/)

4. **Acesse a pasta dos exercícios:**

```bash
cd exercicios
```

5. **Execute qualquer exercício com o Node.js:**

```bash
node exe1.js
```

(Substitua `exe1.js` pelo nome do arquivo do exercício que deseja executar, de `exe2.js` a `exe15.js`)

## 📌 Observações

* A pasta `node_modules` está ignorada no `.gitignore` e não está versionada.
* O projeto é destinado a fins educacionais.
* Sinta-se à vontade para estudar, modificar e testar os exercícios.
